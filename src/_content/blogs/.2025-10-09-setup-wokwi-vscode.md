# ใช้งาน Visual Studio Code ร่วมกับ Wokwi

### ติดตั้ง Extension Wokwi

1. คลิกปุ่ม **รูปสี่เหลี่ยมด้านซ้ายมือ** (ในโปรแกรม Visual Studio Code)
2. พิมพ์ **wokwi** เพื่อค้นหา
3. คลิกปุ่ม **Install** (ของ Extension Wokwi Simulator)

![](https://iili.io/KwquxFn.png)

### ติดตั้ง wokwi License

1. พิมพ์ **F1** (ในโปรแกรม Visual Studio Code)

![](https://iili.io/Kwq5kl9.png)

1. พิมพ์ **Wokwi**
2. เลือก **Request a New License**

![](https://iili.io/Kwq7y7e.png)

1. คลิกปุ่ม **Open**

![](https://iili.io/KwqYvTb.png)

1. คลิกปุ่ม GET YOU LICENSE

![](https://iili.io/KwqlfEl.png)

1. คลิกปุ่ม Open Visual Studio Code

![](https://iili.io/Kwq1ozu.png)

1. คลิกปุ่ม Open (ในโปรแกรม vscode)

### ติดตั้ง PlatformIO IDE Extension

![](https://iili.io/KwqEWCb.png)

1. คลิกปุ่ม **รูปสี่เหลี่ยมด้านซ้ายมือ** (ในโปรแกรม Visual Studio Code)
2. พิมพ์ platformio เพื่อค้นหา
3. คลิกปุ่ม **Install** (ของ Extension PlatformIO IDE)

### สร้างโปรเจค Arduino ด้วย Platform.io

![](https://iili.io/KwqG6xe.md.png)

1. คลิกปุ่ม **รูปมด** (ด้านซ้ายมือ)
2. คลิกปุ่ม **New Project**

![](https://iili.io/KwqV9v2.png)

1. พิมพ์ **ESP32** เป็นชื่อโปรเจค
2. เลือก **Espressif ESP32 Dev Module** (ค้นหาด้วยการพิมพ์ Dev Module)

![](https://iili.io/KwqVPFj.png)

1. ไม่เลือก **Use default location**
2. เลือกที่บันทึกโปรเจคเช่น \*\*D:\*\*

![](https://iili.io/KwqWrbI.png)

1. คลิกปุ่ม **Finish**

### สร้างไฟล์สำหรับ Wokwi

![](https://iili.io/KwqXsTB.png)

1. คลิกปุ่ม **รูปกระดาษ**
2. พิมพ์ **wokwi.toml** เป็นชื่อไฟล์

![](https://iili.io/KwqjBoP.png)

```toml
[wokwi]
version = 1
firmware = '.pio\build\esp32dev\firmware.bin'
elf = '.pio\build\esp32dev\firmware.elf'
```

1. พิมพ์ โค้ดด้านบน ในไฟล์ wokwi.toml

![](https://iili.io/KwqXsTB.png)

1. คลิกปุ่ม **รูปกระดาษ**
2. พิมพ์ **diagram.json** เป็นชื่อไฟล์

![](https://iili.io/KwqOROx.png)

1. คลิกแท็บ diagram.json
2. พิมพ์ Ctrl+A และ Ctrl+C เพื่อก็อปปี้โค้ด

![](https://iili.io/KwqeME7.png)

1. คลิกขวาที่ไฟล์ **diagram.json**
2. เลือก **Open With …**
3. เลือก **Text Editor**

![](https://iili.io/Kwqk2j4.png)

1. พิมพ์ Ctrl+V เพื่อวางโค้ด

### เขียนโค้ดโปรแกรม

![](https://iili.io/Kwqkil9.png)

1. คลิกไฟล์ **main.cpp**
2. เขียนโค้ดโปรแกรม

![](https://iili.io/KwqvXbj.png)

1. คลิกปุ่ม รูปเครื่องหมายถูก เพื่อ Build โปรแกรม

![](https://iili.io/Kwq8uEB.png)

1. คลิกไฟล์ **diagram.json**
2. คลิก **ปุ่มสีเขียว**
